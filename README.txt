----------------------------------------------------------
Aplicación interactiva que permite diseñar compiladores
mediante Diagramas de Tombstone o Diagramas T
----------------------------------------------------------


----------------Controles y atajos:-----------------------

	-Barra de menú-
Nuevo			Ctrl + N
Abrir			Ctrl + A
Guardar			Ctrl + G
Salir			Ctrl + S
Acerca de		Ctrl + D

	-Popup Menu-
Agregar Compilador	Ctrl + C
Agregar Máquina		Ctrl + M
Agregar Programa	Ctrl + P
Agregar Interprete	Ctrl + I
