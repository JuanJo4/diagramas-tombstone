----------------------------------------------------------
Aplicacion interactiva que permite diseñar compiladores
mediante Diagramas de Tombstone o Diagramas T
----------------------------------------------------------


----------------Controles y atajos:-----------------------

	-Barra de menu-
Nuevo			Ctrl + N
Abrir			Ctrl + A
Guardar			Ctrl + G
Salir			Ctrl + S
Acerca de		Ctrl + D

	-Popup Menu-
Agregar Compilador	Ctrl + C
Agregar Maquina		Ctrl + M
Agregar Programa	Ctrl + P
Agregar Interprete	Ctrl + I
